============
Einführung
============

Was macht diese Extension?
========================

Die TYPO3 Hyphenation Extension bietet eine flexible Lösung für die Silbentrennung in TYPO3-Websites. Sie unterstützt sowohl automatische als auch manuelle Silbentrennung und ist besonders nützlich für:

* Präzise Kontrolle über die Silbentrennung von Fachbegriffen
* Verbesserte Lesbarkeit von Texten
* Optimierte Darstellung auf verschiedenen Bildschirmgrößen

Hauptfunktionen
==============

* Manuelle Definition von Trennungspunkten für spezifische Begriffe
* Backend-Modul zur Verwaltung von Trennungsregeln
* ViewHelper für die einfache Integration in Templates
* Unterstützung für mehrsprachige Websites
