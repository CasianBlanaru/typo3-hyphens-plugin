================
Benutzerhandbuch
================

Zielgruppe: **Redakteure**

Silbentrennung im Backend
========================

Trennungsregeln erstellen
------------------------

1. Öffnen Sie das Backend-Modul "Silbentrennung"
2. Klicken Sie auf "Neue Trennungsregel"
3. Geben Sie den Originaltext ein
4. Markieren Sie die Trennungspunkte mit dem Symbol "|"
5. Speichern Sie die Regel

.. figure:: ../Images/User/CreateRule.png
   :class: with-shadow
   :alt: Neue Regel erstellen
   :width: 500px

   Erstellen einer neuen Trennungsregel

Trennungsregeln verwalten
------------------------

* Bestehende Regeln können bearbeitet oder gelöscht werden
* Die Reihenfolge der Regeln kann durch Drag & Drop angepasst werden
* Regeln können temporär deaktiviert werden

Integration in Inhaltselemente
============================

1. Erstellen Sie ein neues Inhaltselement oder bearbeiten Sie ein bestehendes
2. Aktivieren Sie die Option "Silbentrennung verwenden"
3. Wählen Sie die gewünschten Trennungsregeln aus

.. figure:: ../Images/User/ContentElement.png
   :class: with-shadow
   :alt: Inhaltselement-Einstellungen
   :width: 500px

   Silbentrennung in Inhaltselementen aktivieren

Tipps und Best Practices
======================

* Verwenden Sie Trennungsregeln für häufig vorkommende Fachbegriffe
* Testen Sie die Darstellung auf verschiedenen Bildschirmgrößen
* Beachten Sie die Lesbarkeit bei der Definition von Trennungspunkten
